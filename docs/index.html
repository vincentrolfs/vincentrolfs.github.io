About me

Hi, I'm Vincent! I am a senior-level software engineer. That means I am comfortable and experienced with complex development tasks such as writing parsers, performing security audits, improving software architecture, formulating long-term test strategies, and more.

I am well-versed in TypeScript, Kotlin, PHP, Python, Java and SQL. I am also open to learning new programming languages for a particular project and have done so in the past. At the moment, I am focusing on learning Rust through my side projects.

In my work, I strive for Holistic Engineering: I can help you plan, implement and support the entire software system from beginning to end. This includes technical aspects like the system’s frontend, backend, database and mobile apps, but also social aspects such as understanding the user’s needs and communicating concepts to non-technical people.

I speak German and English fluently, as well as some Mandarin and French.

Feel free to message me if you are interested in working together.
